# hjkleroids

Very small Python game.

Made mainly to learn Vim, so you move with **hjkl**.
You also shoot with `;`.

The bullets are circles because collision is hard.  
You can go off screen to keep you on your toes (and because I got bored).
If I knew what I was doing (or had an LSP), there'd be more here.
